format wysyłania RFID od raspberry do backendu 
topic: rfid
payload: zseralizowany json
{
    rfid: 123456,
    time: 2022-03-15 12:30:45
}


aby uruchomić brokera należy wpisać poniższe instrukcje
sudo nano /etc/mosquitto/mosquitto.conf

dopisać

allow_anonymous true
listener 1883 0.0.0.0

sudo systemctl start mosquitto.service
